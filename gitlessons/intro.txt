6. git pull <orugin> <branch> - команда при помощи которой можно стягивать изменения с удаленного репозитория.

7. git clone < url> - команда для клонирования удаленного репозитория и всего его содержимого.

8. git branch < name > - команда для создания новой ветки. 

9. git checkout < name > - команда для переключения между ветками.

10. git revert - команда для возвращения на опреденный коммит , соответственно для восстановления предыдущих версий кода.
git revert < ссылка на коммит>

git log - можно посмотреть историю всех коммитов и посмотреть ссылки на коммиты.
